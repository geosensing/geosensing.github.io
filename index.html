geosensing

The project is about making it easier to collect and analyze geographically distributed data. We showcase the tools with some interesting applications.

Software

  • geo_sampling: Randomly sample locations on the streets. Python Package | Documentation
  • allocator: Efficiently collect data from geographically distributed locations. Python Package | Documentation
  • ybar: Mobile application for crowd sourcing data collection over geographically distributed locations. Architecture And Application
  • Implementation of Christofides Algorithm in Python for TSP. Script.

Applications

  • Women count: Missing Women on the streets of Delhi. Project
  • Street Sense: Learning from Google Street View. Project
